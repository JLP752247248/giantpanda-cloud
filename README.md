# giantpanda-cloud
项目主旨：
尽量整合更多的技术栈，既可以作为一个全栈demo，也可以作为一个框架搭建的脚手架使用

已有功能：
用户管理
角色管理
权限管理
文件服务
微信小程序

已用技术：
SpringCloud/springMVC/Spring-security/JPA+Hibernate
Nacos
腾讯云Cos
Vue3.0
Flutter
sentinel
